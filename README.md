# Travel SDK

## Development

### Building the SDK

```bash
npm run build
```

### Running the widget

```bash
npm run widget
```

### Watch mode for development

In one terminal:
```bash
npm run dev  # watch and rebuild SDK on changes
```

In another terminal:
```bash
npm run widget  # run the widget
```
